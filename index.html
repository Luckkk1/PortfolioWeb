사용 언어: HTML, CSS, Bootstrap

깃헙 바로가기
  • CSS 이용을 최소화하고 Bootstrap을 이용했습니다.
  • Bootstrap의 다양한 기능을 구현하였습니다.
  • px 보단 Rem, em, 그리고 %를 이용하여 어떤 화면사이즈에서도 원하는 화면이 사용자에게 구현될 수 있게 하였습니다.

포트폴리오 웹사이트

사용 언어: HTML, CSS, JavaScript

깃헙 바로가기
  • CSS 글로벌 변수를 설정하여 자주 사용하는 값을 재사용하였습니다.
  • 반응형 웹으로 만들어 어떠한 크기의 화면에서도 컨텐츠가 잘 출력되도록 하였습니다.
  • IntersectionObserver Web API를 활용하여 보고 있는 화면을 감지하여 해당하는 메뉴에 강조표시하는 기능을 구현하였습니다.
  • ScrollintoView를 활용하여 메뉴, 애로우업버튼을 누르면 해당 섹션으로 이동이 가능합니다.
